# Actieplan — van "site klaar" naar "bovenaan scoren" (30 dagen)

> On-site SEO is gemaximaliseerd (73 pagina's, 288 schema-blokken, 0 fouten, mobile-first, IndexNow). Ranking #1 is nu een functie van **indexering + off-page autoriteit + tijd**. Dit plan zet die drie in de optimale volgorde. Tijd per item staat erbij; de meeste zijn 10-30 min.

## Waarom een plan i.p.v. nog meer pagina's
De content-queue is af. Méér pagina's bouwen vóór er zoekdata is = gokken = cannibalisatie-risico. De juiste volgorde is: **eerst indexeren + autoriteit + meten**, dán pagina's bijbouwen op basis van échte query-data uit Search Console (zie Week 4).

---

## DAG 1-2 — Fundament (kritiek, doe dit eerst)
- [ ] **Google Search Console** aanzetten + property verifiëren (`01-gsc-bing-setup.md`). Zonder dit zie je geen Google-data.
- [ ] In GSC: **`sitemap.xml` indienen** (+ `sitemap-news.xml`, `sitemap-images.xml`). Google gebruikt geen IndexNow, dus dit is de enige manier om indexering te triggeren.
- [ ] **Bing Webmaster Tools** aanzetten (zelfde sitemap). Bing pakt IndexNow al op — dit versterkt het.
- [ ] GSC → URL-inspectie → "Indexering aanvragen" voor de 5 belangrijkste pagina's: `/`, `/prijzen`, `/robot-as-a-service`, `/huren`, `/robots`.

## WEEK 4 — METEN → data-gedreven volgende ronde on-site
Dit is waar de cyclus terugkomt naar mij/de site, nu **niet meer speculatief**:
- [ ] GSC → **Prestaties → Zoekopdrachten**: exporteer de queries waarop je impressies krijgt.
- [ ] Markeer queries met **veel impressies maar lage CTR / positie 8-20** = bewezen vraag waar je net niet scoort.
- [ ] **Dán** pas nieuwe pagina's bouwen / bestaande verdiepen — gericht op díe queries. Dit is de enige veilige manier om door te bouwen zonder cannibalisatie.
- [ ] Titel/meta van pagina's met hoge impressie + lage CTR herschrijven (snelle CTR-winst).

## Wat te verwachten (realistisch, jong domein)
| Termijn | Verwachting |
|---|---|
| 2-5 dagen | Pagina's geïndexeerd (versneld via IndexNow + GSC) |
| 2-3 weken | Eerste organic impressies meetbaar in GSC |
| 4-8 weken | Long-tail rankings (lange, specifieke queries) komen op |
| 3-6 maanden | Head-terms stijgen — afhankelijk van backlinks/autoriteit |

"Altijd #1 op alle termen" is geen schakelaar; het is het resultaat van bovenstaande, herhaald. De head-terms ("humanoïde robot leasen") win je pas ná autoriteitsopbouw — de niche is open (geen dominante concurrent), dus het is haalbaar, maar het kost weken-maanden, niet uren.

## Doorlopend (de enige veilige "blijf bouwen"-modus)
- News-bot blijft draaien → freshness-signaal (al actief).
- Maandelijks GSC-review → nieuwe pagina's op bewezen queries (Week 4-cyclus herhalen).
- Backlinks blijven aanvragen wanneer er nieuws/mijlpalen zijn.
