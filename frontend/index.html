Hoe werkt BotLease?

Van aanmelding tot live chatbot in 4 simpele stappen

1

Wij bouwen jouw chatbot

Jij levert informatie over je bedrijf, producten en veelgestelde vragen. Wij trainen de AI met al deze kennis zodat de chatbot perfect jouw bedrijf kan vertegenwoordigen.

2

Je krijgt een simpel script

Geen technische kennis nodig! Je krijgt van ons een klein stukje code dat je kopieert en plakt op je website. Werkt met elke website builder.

3

Direct live op je website

Zodra het script op je website staat, verschijnt de chatbot automatisch. Klanten kunnen meteen vragen stellen en krijgen slimme antwoorden.

4

Beheer via je dashboard

In je persoonlijke dashboard zie je alle gesprekken, veelgestelde vragen en statistieken. Je kunt de chatbot blijven verbeteren en eventueel koppelen aan je team.
